A simple plugin that allows you to create a widget showing a list of posts of a custom post type.

== Description ==

This plugin allows you to create a widget that displays a number of posts from a previously defined custom post type.

**This plugin does not create any custom post types!**

The widget options allow you to provide a custom title, choose which post type, what order to show them in, and 
whether or not to show the excerpt.
  
== Installation ==

1. Upload `/ctp-widget/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the Widget panel to place and configure this widget in a sidebar
